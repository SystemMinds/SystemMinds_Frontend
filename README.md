# Apple Website Clone

A complete React clone of the Apple India website built with Vite, React (JSX), and Tailwind CSS.

## Features

- ✅ Complete UI recreation of Apple.com/in
- ✅ Responsive design for mobile, tablet, and desktop
- ✅ Modern React components using JSX
- ✅ Tailwind CSS for styling
- ✅ Vite for fast development and building

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```
